# 📝 Smart To-Do App

A modern To-Do List web app built using React.

## 🚀 Features

* Add & delete tasks
* Mark tasks as completed ✔️
* Edit tasks ✏️
* Search tasks 🔍
* Dark / Light mode 🌙☀️
* LocalStorage support 💾
* Responsive design 📱
* Glassmorphism UI ✨

## 🛠 Tech Stack

* React (Vite)
* JavaScript
* CSS
